# カード文字起こしタイピングゲーム

🃏 カードゲームのOCR結果を人力で修正するためのタイピングゲームです。

## 🎮 プレイ方法

### GitHub Pagesでプレイ

**URL: https://omezi42.github.io/AnokoroCardsDictionary/**

> ⚠️ `index.html`と`card_database.json`と`card_images/`が同じリポジトリにあることを確認してください。

### ローカルでプレイ

1. このリポジトリをクローンしてください
2. `index.html`をブラウザで開いてください
3. 画像とOCR結果が表示されたら、正しいテキストを入力して「保存して次へ」をクリック

## ✨ 機能

- **画像表示**: カード画像をフィールドごとに表示
- **OCR比較**: 現在のOCR結果と入力テキストを色分け表示
- **進捗保存**: ブラウザに自動保存（localStorage）
- **複数フィールド対応**: name, effect, lore, yomi, type, power, cost, set, tag
- **エクスポート**: JSON/CSV形式で修正データをエクスポート
- **インポート**: 以前保存したJSONを読み込んで続きから再開
- **キーボードショートカット**:
  - `←` `→`: カード移動
  - `Enter`: 入力エリアにフォーカス

## 📁 ファイル構成

```
.
├── index.html           # タイピングゲーム本体
├── card_database.json   # OCRで文字起こしされたデータ
├── card_images/         # カード画像フォルダ
│   ├── name/            # カード名画像
│   ├── effect/          # 効果画像
│   ├── lore/            # 説明文画像
│   ├── yomi/            # 読み画像
│   ├── type/            # タイプ画像
│   ├── power/           # パワー画像
│   ├── cost/            # コスト画像
│   ├── set/             # セット画像
│   └── tag/             # タグ画像
└── README.md
```

## 🔧 使い方

1. **フィールド選択**: 「name」「effect」「lore」などをドロップダウンから選択
2. **画像を確認**: 上部の画像を見て正しい文字を読んでください
3. **テキスト入力**: 下のテキストエリアに正しいテキストを入力してください
4. **保存**: 「保存して次へ」をクリックして次のカードへ移動
5. **エクスポート**: すべての修正が完了したら「JSON出力」で修正データをダウンロード

## 💾 データ保存

- **自動保存**: 入力内容はブラウザのlocalStorageに自動保存されます
- **エクスポート**: 作業完了後にJSON/CSVとして保存することを推奨
- **インポート**: 保存したJSONを読み込んで作業を再開できます

## 🤝 コントリビュート

修正が完了したら、修正データをJSONでエクスポートしてPull Requestを送ってください！
